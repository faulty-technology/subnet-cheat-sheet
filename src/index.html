Formulas

Number of subnets: 2n where n is the number of borrowed bits

Number of hosts: 2(32-n) - 2 where n is the number of subnet mask bits

Block size: The block size for a subnet is 256 - subnet mask value

Broadcast address: The broadcast address is one less than the next network address

Usable hosts: The usable hosts are between the network address and the broadcast

* /31 networks are special point-to-point links (RFC 3021) where both addresses are usable
